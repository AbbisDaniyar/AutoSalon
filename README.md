# AutoSalon

Простое приложение «Автосалон» на Node.js + Handlebars.

## Краткое описание
Веб-приложение для управления каталогом автомобилей, клиентами и продажами.

## Требования
- Node.js (v14+ рекомендуется)
- npm

## Установка
В корне проекта выполните:
```bash
npm install
```

## Переменные окружения
Создайте файл `.env` в корне проекта
```
PORT=3000
DB_HOST=localhost
DB_USER=your_user
DB_PASS=your_password
DB_NAME=AutoSalon
```

## Запуск
Запустить сервер можно напрямую:
```bash
node app.js
```

После запуска откройте в браузере: `http://localhost:3000` (или порт из `PORT`).

## База данных
Используется MySQL DB в качестве основной. SQL-скрипт для создания таблиц находится в файле:
- `database.sql`

## Структура проекта (основное)
- `app.js` — точка входа
- `controllers/` — обработчики маршрутов
- `routes/` — маршруты
- `views/` — Handlebars шаблоны (layouts в `views/layouts`)
- `public/` — статические файлы (CSS, JS, изображения)
